# Changelog

All notable changes to this project will be documented in this file.

## [1.18.3-alpha.1] - 2025-11-28

### Added
- Initial release of CROSSx WebApp SDK
- `CROSSx.WebApp.version` - Get SDK version
- `CROSSx.WebApp.ready()` - Signal app readiness
- `CROSSx.WebApp.requestFullScreen()` - Request fullscreen mode
- `CROSSx.WebApp.on()` - Register lifecycle event listeners
- `CROSSx.WebApp.off()` - Remove event listeners
- Support for `viewClosed` event - triggered when WebView is closed
- Support for `viewBackgrounded` event - triggered when app moves to background
- Mock implementation for development and testing
- Environment detection for CROSSx vs browser environments
- Native bridge communication layer
- Full TypeScript support
- Comprehensive documentation and examples
